Manual de Uso — Turnia

Guía completa para gestionar tu agenda de citas desde el panel del negocio.

📅 Última actualización: 21 de mayo 2026 📖 Para personas no técnicas

Perfiles

Turnia tiene tres perfiles diferenciados según el rol de quien accede:

🏪 Negocio / Admin

  • Ver y crear citas
  • Confirmar, cancelar, reprogramar
  • Gestionar clientes y servicios
  • Configurar profesionales
  • Ver reportes y exportar

👤 Cliente final

  • Entrar al link público
  • Elegir servicio y profesional
  • Seleccionar fecha y horario
  • Enviar reserva con sus datos

⚠️ Sin negocio asignado

  • Puede iniciar sesión
  • Ve un aviso informativo
  • No accede al panel hasta ser asignado

Pantalla de inicio

Desde la pantalla de inicio tienes tres opciones:

  • Entrar como negocio — abre el panel de gestión del negocio
  • Entrar con Google — en demo simula login; en producción usará Google
  • Ver link del cliente — muestra la experiencia pública de reserva

Agenda

La agenda muestra citas y bloqueos del día seleccionado. Controles disponibles: selector de fecha, filtro por profesional, botón Nueva cita y botón Bloquear horario.

Estados de cita

Pendiente Confirmada Finalizada Cancelada No asistió

Crear cita manual

  1. Entrar al panel del negocio
  2. Pulsar Nueva cita
  3. Cargar cliente y teléfono
  4. Elegir servicio y profesional
  5. Elegir fecha y hora
  6. Agregar nota si hace falta
  7. Guardar
Turnia valida automáticamente si el horario elegido está libre antes de guardar.

Confirmar cita

  1. Buscar una cita pendiente
  2. Pulsar Confirmar
  3. Turnia cambia el estado a Confirmada
  4. Se prepara mensaje de WhatsApp para enviar al cliente

Cancelar cita

  1. Buscar la cita
  2. Pulsar Cancelar
  3. Turnia cambia el estado
  4. Se prepara mensaje para avisar al cliente

Reprogramar cita

  1. Pulsar Reprogramar
  2. Elegir nueva fecha
  3. Elegir profesional
  4. Elegir horario disponible
  5. Agregar nota si hace falta
  6. Guardar
Turnia evita elegir horarios ocupados o bloqueados al reprogramar.

Finalizar cita

  1. Buscar una cita confirmada
  2. Pulsar Finalizar
  3. La cita queda cerrada en el historial

Marcar no asistencia

  1. Buscar una cita confirmada
  2. Pulsar No asistió
  3. El historial del cliente queda marcado

Bloqueos

Los bloqueos impiden reservas en un horario específico. Útiles para: comida, descanso, ausencia, tarea interna o profesional no disponible.

Crear bloqueo

  1. Ir a Agenda
  2. Pulsar Bloquear horario
  3. Elegir fecha y profesional
  4. Indicar inicio y fin
  5. Escribir motivo
  6. Guardar
El link del cliente no mostrará ese horario como disponible mientras exista el bloqueo.

Vista semanal

Permite revisar la semana completa de un vistazo.

  1. Entrar a Semana
  2. Elegir profesional
  3. Revisar citas y bloqueos por día

Clientes

Crear cliente

  3. Cargar nombre, teléfono y nota
  4. Guardar

Editar cliente

  1. Seleccionar el cliente
  2. Editar desde su ficha
  3. Guardar

Servicios

Los servicios son la base del sistema. Turnia usa la duración de cada servicio para calcular si un horario entra completo en la agenda.

Campos de un servicio

  • Nombre — cómo lo ve el negocio o el cliente. Ej: Color + corte
  • Categoría — ayuda a ordenar. Ej: Peluquería, Uñas, Estética
  • Duración — minutos que ocupa el turno. Clave para evitar superposiciones
  • Precio — importe estimado del servicio, usado en reportes
  • Profesionales — personas que pueden realizar ese servicio
  • Visible online — define si aparece en el link público del cliente

Crear servicio

  1. Ir a Servicios
  2. Pulsar Nuevo servicio
  3. Completar nombre, categoría, duración y precio
  4. Elegir profesionales que pueden realizarlo
  5. Marcar visibilidad online
  6. Guardar
Ejemplo práctico: Servicio "Color + corte", 90 min, 58€, profesional: Mara, visible online: sí. El cliente solo verá horarios donde Mara tenga 90 minutos libres.

Eliminar servicio

No se puede eliminar un servicio que ya tiene citas asociadas, para evitar romper historiales.

Equipo

Cada profesional tiene nombre, especialidad, horario de trabajo y servicios asignados. Turnia usa estos datos para calcular disponibilidad.

Crear profesional

  1. Ir a Equipo
  2. Pulsar Nuevo profesional
  3. Escribir nombre y especialidad
  4. Indicar hora de inicio y fin
  5. Seleccionar servicios que puede realizar
  6. Guardar
Ejemplo: Noe, Manicura y barbería, horario 10:00–19:00. Solo aparecerá disponible para los servicios asignados dentro de su horario libre.

Relación servicios + equipo

Turnia calcula disponibilidad combinando: horario del negocio, horario del profesional, duración del servicio, citas existentes, bloqueos y servicios asignados.

Link cliente

El Link Cliente es la vista pública que el negocio puede poner en Instagram, Google, WhatsApp o su web. Los clientes reservan desde ahí sin necesidad de llamar.

Acceder al link

  • Pantalla inicial → Ver link del cliente
  • Panel negocio → Vista cliente
  • Menú lateral → Link cliente

Flujo de reserva del cliente

  1. Cliente entra al link
  2. Elige servicio
  3. Elige profesional (o "cualquiera disponible")
  4. Elige fecha
  5. Turnia muestra horarios libres
  6. Cliente selecciona horario
  7. Carga nombre, teléfono y nota
  8. Envía reserva
Si el cliente elige "Cualquiera disponible", Turnia busca cualquier profesional que pueda hacer ese servicio y tenga horario libre.

Después de reservar

Si la configuración es de confirmación manual, la cita entra como Pendiente. Con auto-confirmación activa, entra directamente como Confirmada.

Reportes

Los reportes permiten revisar la actividad del negocio: cantidad de turnos, ingresos estimados, ticket medio, estados (pendientes, finalizadas, no asistió, canceladas), servicios más solicitados e ingresos por profesional.

Botón Guardar PDF: usa la función imprimir/guardar como PDF del navegador.

Configuración

Permite personalizar el negocio y sus reglas de funcionamiento.

Datos del negocio

  • Nombre y slug del link
  • Ciudad y teléfono
  • Horario general

Reglas de reserva

  • Intervalo de reservas
  • Anticipación mínima
  • Auto-confirmación

Comunicación

  • Plantillas de WhatsApp

Exportar datos

  • Clientes Excel
  • Citas Excel
  • Backup JSON

Restaurar demo

El botón Restaurar demo borra los cambios guardados en el navegador y vuelve a los datos iniciales. Usarlo solo cuando se quiera limpiar la demo para una presentación.

También se puede abrir la URL con ?reset-demo=1 para limpiar la demo antes de una presentación.
