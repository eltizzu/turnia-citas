# Turnia

## Agenda online simple para negocios que trabajan por turnos

Turnia es una herramienta para ayudar a salones, barberias, centros de estetica y profesionales de servicios a organizar sus citas, reducir mensajes repetidos y permitir que sus clientes reserven desde un link.

La idea es simple: el negocio mantiene el control de su agenda y el cliente puede pedir turno de forma clara, sin idas y vueltas innecesarias.

## El problema

Muchos negocios gestionan sus turnos con WhatsApp, llamadas, notas sueltas o una agenda manual.

Eso suele generar:

- clientes preguntando una y otra vez que horarios hay disponibles;
- turnos pisados o mal anotados;
- dificultad para saber quien esta libre y cuando;
- poco control sobre cancelaciones o cambios;
- falta de historial del cliente;
- poca claridad sobre ingresos generados por las citas;
- demasiado tiempo administrativo durante el dia.

Turnia busca ordenar ese proceso sin obligar al negocio a usar un sistema complicado.

## Para quien esta pensado

Turnia esta pensado para negocios que trabajan con reservas por horario, especialmente:

- peluquerias;
- barberias;
- centros de estetica;
- manicura y unas;
- pestanas y cejas;
- masajes;
- profesionales independientes;
- estudios pequenos con varios profesionales.

Es ideal para negocios que reciben clientes por Instagram, WhatsApp, Google o recomendacion, y quieren ofrecer una forma mas ordenada de reservar.

## Que permite hacer

### Agenda del negocio

El negocio puede ver sus citas por dia, revisar pendientes, confirmar turnos, cancelar, reprogramar, finalizar citas o marcar cuando un cliente no asistio.

### Link de reserva para clientes

Turnia permite tener una vista para el cliente donde puede:

- elegir fecha;
- elegir servicio;
- elegir profesional o cualquiera disponible;
- ver horarios libres;
- cargar sus datos;
- enviar la reserva.

Ese link puede compartirse en Instagram, WhatsApp, Google Business Profile o una web.

### Servicios y precios

El negocio puede cargar sus servicios con:

- nombre;
- categoria;
- duracion;
- precio;
- profesionales que lo realizan;
- visibilidad online.

La duracion del servicio ayuda a que Turnia muestre horarios reales y evite superposiciones.

### Profesionales

Cada profesional puede tener:

- nombre;
- especialidad;
- horario de trabajo;
- servicios que puede realizar.

Asi Turnia puede responder preguntas como:

- que profesional esta libre a una hora;
- quien puede hacer determinado servicio;
- que horarios quedan disponibles.

### Clientes

Turnia permite tener una ficha simple de cada cliente:

- nombre;
- telefono;
- notas;
- historial de citas;
- acceso rapido a WhatsApp.

Esto ayuda a dar un trato mas ordenado y recordar preferencias importantes.

### Bloqueos de horario

El negocio puede bloquear horarios por profesional para:

- descansos;
- comida;
- ausencias;
- tareas internas;
- vacaciones cortas.

Los horarios bloqueados no aparecen disponibles para el cliente.

### Reportes

Turnia muestra informacion basica para entender el movimiento del negocio:

- cantidad de turnos;
- ingresos estimados;
- ticket medio;
- citas pendientes;
- citas finalizadas;
- no asistencias;
- servicios mas solicitados;
- ingresos por profesional.

### Exportaciones

El negocio puede descargar:

- clientes en Excel;
- citas en Excel;
- backup tecnico en JSON.

Esto refuerza una idea importante: los datos del negocio siguen siendo del negocio.

## Valor para el negocio

Turnia ayuda a:

- ahorrar tiempo contestando disponibilidad;
- reducir errores de agenda;
- ordenar servicios y profesionales;
- evitar turnos superpuestos;
- tener clientes e historial en un solo lugar;
- confirmar o reprogramar con mas claridad;
- entender mejor el movimiento del mes;
- ofrecer una experiencia mas profesional al cliente.

## Valor para el cliente final

El cliente puede:

- ver opciones disponibles;
- elegir servicio;
- elegir profesional si quiere;
- reservar sin esperar respuesta inmediata;
- dejar una nota;
- recibir confirmacion del negocio.

La reserva se vuelve mas simple y mas clara.

## Diferenciales de Turnia

- No obliga a entrar en un marketplace.
- No cobra comision por cada reserva.
- Mantiene WhatsApp como canal natural del negocio.
- Esta pensado para negocios no tecnicos.
- Prioriza una experiencia simple desde el movil.
- Permite un setup inicial asistido.
- Se adapta bien a primeros clientes con configuracion personalizada.

## Como se podria implementar en un negocio

Para un primer piloto, el proceso recomendado seria:

1. Relevar servicios, precios y duraciones.
2. Cargar profesionales y horarios.
3. Configurar reglas de reserva.
4. Revisar el link cliente.
5. Probar reservas internas.
6. Compartir el link con algunos clientes.
7. Medir si reduce mensajes y ordena la agenda.

## Estado actual del producto

Turnia esta en una demo funcional lista para mostrar y validar con primeros negocios.

La demo permite ver el flujo completo:

- panel del negocio;
- agenda;
- servicios;
- profesionales;
- clientes;
- link de reserva;
- reportes;
- exportaciones.

Para usarlo en produccion con datos reales, el siguiente paso es conectarlo a una base de datos en la nube, login real, hosting, dominio y revision legal/fiscal.

## Modelo comercial inicial posible

Para primeros pilotos, una opcion simple seria:

- setup inicial para cargar servicios, profesionales y reglas;
- mensualidad fija;
- sin comisiones por reserva.

Ejemplo a validar:

- setup inicial segun complejidad;
- mensualidad segun cantidad de profesionales;
- soporte cercano durante el piloto.

## Mensaje corto para presentar

Turnia ayuda a negocios de servicios a ordenar su agenda y permitir que sus clientes reserven desde un link, manteniendo el control del negocio y reduciendo el tiempo perdido en mensajes repetidos.

## Cierre

Turnia busca ser una agenda moderna, clara y practica para negocios que necesitan ordenar sus turnos sin sumar complejidad.

El objetivo del primer piloto no es prometer automatizacion total, sino validar algo mas concreto: que el negocio ahorre tiempo, evite errores y pueda ofrecer una experiencia de reserva mas profesional.
