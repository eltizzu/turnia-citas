# Checklist De Setup Para Primer Cliente

Este checklist sirve para entregar Turnia casi personalizado a un negocio real.

## 1. Datos Del Negocio

- Nombre comercial.
- Ciudad.
- Telefono de contacto.
- Instagram.
- Web, si tiene.
- Direccion, si se mostrara en el link publico.
- Logo o color principal, si se quiere personalizar.

## 2. Horarios

- Dias de apertura.
- Hora de apertura y cierre.
- Descansos o cortes.
- Feriados o dias especiales.
- Anticipacion minima para reservar.
- Intervalo de turnos: 15, 30 o 60 minutos.

## 3. Profesionales

Por cada profesional:

- Nombre.
- Especialidad.
- Horario propio, si difiere del negocio.
- Servicios que puede realizar.
- Ausencias conocidas.

En la demo actual se cargan desde:

Turnia -> Equipo -> Nuevo profesional.

## 4. Servicios

Por cada servicio:

- Nombre.
- Categoria.
- Duracion.
- Precio.
- Profesionales que lo realizan.
- Si se puede reservar online.
- Margen extra antes o despues, si aplica.

En la demo actual se cargan desde:

Turnia -> Servicios -> Nuevo servicio.

## 5. Reglas De Reserva

- Las reservas online entran pendientes o confirmadas.
- Se permite elegir profesional o solo "cualquiera disponible".
- Se permite cancelar o pedir cambio desde el link.
- Tiempo minimo antes del turno.
- Si el negocio quiere aprobar manualmente todo al principio.

## 6. Bloqueos De Agenda

Cargar bloqueos cuando el negocio no quiere recibir citas:

- Comidas o descansos.
- Reuniones.
- Ausencias parciales.
- Vacaciones.
- Formaciones.
- Horarios reservados para tareas internas.

En la demo actual se cargan desde:

Turnia -> Agenda -> Bloquear horario.

## 7. Clientes

Si el negocio ya tiene una base simple de clientes, cargar primero:

- Nombre.
- Telefono.
- Nota interna util.
- Preferencias relevantes.

En la demo actual se cargan desde:

Turnia -> Clientes -> Nuevo cliente.

## 8. WhatsApp

Preparar textos para:

- Confirmacion.
- Cancelacion.
- Reprogramacion.
- Recordatorio.
- Mensaje para pegar en Instagram o WhatsApp Business.

Ejemplo para perfil:

Reserva tu turno online aca:
{link_publico}

## 9. Legal Basico

Antes de usar con clientes reales:

- Adaptar politica de privacidad.
- Informar uso del telefono para gestionar la cita.
- No usar datos para marketing sin consentimiento.
- Evitar notas sensibles si no son necesarias.
- Revisar contrato de encargado del tratamiento.

## 10. Prueba De Entrega

- Crear una reserva desde el link publico.
- Confirmarla desde la vista negocio.
- Abrir WhatsApp con mensaje preparado.
- Verificar que el horario deje de aparecer disponible.
- Crear cliente manual.
- Revisar ficha e historial del cliente.
- Crear bloqueo y verificar que el hueco no aparezca en el link publico.
- Ver reporte mensual.
- Guardar reporte en PDF.

## 11. Entrega Al Cliente

Entregar:

- Link publico.
- Acceso al panel.
- Servicios cargados.
- Profesionales cargados.
- Clientes iniciales cargados, si aplica.
- Plantillas de WhatsApp.
- Breve explicacion de agenda, reservas pendientes y reportes.
