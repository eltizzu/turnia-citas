# Idea Del Producto Turnia

## 1. Que Es

Turnia es una app para gestionar citas y agenda en negocios que trabajan por turno u horario.

Esta pensada para:

- Peluquerias
- Barberias
- Unas
- Estetica
- Masajes
- Centros de belleza
- Servicios similares

## 2. Problema Que Resuelve

Muchos negocios siguen mezclando:

- WhatsApp
- Llamadas
- Papel
- Agenda manual
- Recordatorios informales
- Mensajes de Instagram

Eso hace que:

- Se pierdan citas.
- Se solapen horarios.
- Cueste ver huecos libres.
- Se olviden confirmaciones.
- El negocio dependa demasiado de una persona.
- Los clientes no tengan una forma simple de reservar o cambiar horario.

## 3. Version Base

### Gestion Del Negocio

- Cargar nombre del negocio
- Cargar horarios de atencion
- Configurar profesionales
- Configurar servicios

### Agenda

- Ver el dia
- Ver la semana
- Ver huecos disponibles
- Crear cita
- Editar cita
- Cancelar cita
- Reprogramar cita

### Clientes

- Nombre
- Telefono
- Notas
- Historial basico

### Citas

- Servicio
- Profesional
- Fecha
- Hora
- Duracion
- Estado
- Observaciones

### Estados

- Pendiente
- Confirmada
- Cancelada
- Finalizada
- No asistio

## 4. Diferenciacion

Turnia debe diferenciarse por:

- Simpleza
- Movil primero
- Sin comisiones
- Sin marketplace obligatorio
- WhatsApp como canal natural
- Precio facil de entender
- Onboarding rapido para negocios no tecnicos

## 5. Funciones Futuras Importantes

- Recordatorios automaticos
- Confirmacion por WhatsApp
- Reprogramacion desde mensaje
- IA para responder consultas frecuentes
- Estadisticas de ocupacion
- Bloqueo de horarios
- Pagos o senales
- Version multi-sede

## 6. Frase Guia

La agenda simple para salones que quieren ordenar WhatsApp, confirmar citas y llenar huecos sin pagar comisiones.
