## Datos Tratados

Turnia puede tratar:

- Nombre.
- Telefono.
- Servicio solicitado.
- Fecha y hora de la cita.
- Profesional asignado.
- Notas proporcionadas por el cliente.
- Estado de la cita.

## Finalidades

Los datos se usan para:

- Gestionar reservas y citas.
- Confirmar, cancelar o reprogramar turnos.
- Enviar comunicaciones operativas relacionadas con la cita.
- Mantener un historial basico del cliente para el negocio.
- Mejorar la organizacion interna de la agenda.

## Base Juridica

La base juridica podra ser:

- Ejecucion de una solicitud o relacion precontractual/contractual cuando el cliente pide una cita.
- Interes legitimo para comunicaciones operativas necesarias sobre la reserva.
- Consentimiento cuando se trate de comunicaciones comerciales o usos no necesarios.

Debe revisarse para cada caso concreto.

## Destinatarios

Los datos pueden ser tratados por:

- El negocio que recibe la reserva.
- Turnia como proveedor tecnologico.
- Proveedores tecnicos necesarios para alojamiento, comunicaciones o soporte.

No se venderan datos personales a terceros.

## Conservacion

Los datos se conservaran durante el tiempo necesario para gestionar la cita, mantener el historial basico del cliente y cumplir obligaciones legales.

El negocio debera definir su plazo concreto de conservacion.

## Derechos

La persona interesada puede solicitar:

- Acceso.
- Rectificacion.
- Supresion.
- Oposicion.
- Limitacion.
- Portabilidad cuando aplique.
